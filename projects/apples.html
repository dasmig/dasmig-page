Desenvolvido utilizando OpenGL
Fundamentos de Computação Gráfica - INF01047

Sobre

Esse é um trabalho que foi realizado no primeiro semestre de 2016 para a disciplina de Fundamentos de Computação Gráfica. A qual faz parte do curso de Ciência da Computação, do Instituto de Informática da UFRGS.

O trabalho consiste no desenvolvimento de um jogo baseado no clássico DigDug II. Foi feito um menu para o jogo, um carregador de fases, possibilitando ao usuário desenvolver fases adicionais, e o jogo em si.

O desenvolvimento foi feito na linguagem C e foi utilizado a API OpenGL para a parte gráfica. Abaixo encontram-se links para download do relatório de desenvolvimento juntamente com o manual do jogo, o download do executável para Windows, e o download do código fonte.
