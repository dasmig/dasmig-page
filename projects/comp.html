Da Análise Léxica à Otimização
Compiladores - INF01147

Sobre

Esse é um trabalho que foi realizado no primeiro semestre de 2018 para a disciplina de Compiladores. A qual faz parte do curso de Ciência da Computação, do Instituto de Informática da UFRGS.

O trabalho consiste no desenvolvimento de um compilador passando pelas etapas de análise léxica, análise sintática, análise semântica, geração de código para máquina e otimização.

O desenvolvimento foi feito na linguagem C, utilizando FLEX para análise léxica e YACC para análise semântica. O código está disponível em um repositório público do Github.
