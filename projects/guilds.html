Jogo de Estratégia 2D
C++, SFML

Sobre

O jogo começou como um projeto pessoal, mas devido a escala, diversas pessoas começaram a fazer parte do projeto. Hoje conta com 3 programadores e 1 artista.

O jogo está em desenvolvimento, está sendo utilizado o Visual Studio 17 como IDE. Nenhuma engine está sendo utilizada, apenas uma biblioteca para o carregamento de imagens.

A linguagem escolhida foi C++ e a API Simple Fast Multimedia Library está sendo utilizada como interface para componentes multímidias. O código já conta com mais de 30.000 linhas, entretanto o repositório do github está configurado como privado. Para ver o código é necessário enviar um e-mail solicitando.
